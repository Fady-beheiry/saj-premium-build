# SAJ Premium — Construction & Finishing Website

> A high-end, conversion-focused website built for SAJ Construction & Finishing.
> Live demo: [saj-premium.vercel.app](https://saj-premium.vercel.app)

---

## 🧱 Tech Stack

| Layer | Tech |
|-------|------|
| Framework | React + Vite |
| Language | TypeScript |
| Styling | Tailwind CSS |
| UI Components | shadcn/ui |
| Deployment | Vercel |
| Builder | Lovable.dev |

---

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or bun

### Installation

```bash
git clone https://github.com/Fady-beheiry/saj-premium-build.git
cd saj-premium-build
npm install
npm run dev
```

### Build for Production

```bash
npm run build
npm run preview
```

## 📦 Deployment

Auto-deployed via **Vercel** on every push to `main`.

To deploy manually:

```bash
npm run build
# Upload /dist to any static hosting
```
